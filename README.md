##  Approach & Methodology

Here's how I'm tackling this course:

- **Exercises** — Done weekly alongside lectures, focused on applying each concept right after it's introduced.
- **Project** — Developed iteratively, starting with a solid HTML/CSS structure, then adding dynamic behavior with JavaScript, and finally connecting a PHP backend with a database.
- **Code quality** — Keeping code clean, well-commented, and following semantic HTML and separation of concerns (HTML for structure, CSS for style, JS for behavior).
- **Version control** — Committing often with descriptive messages, using branches for separate features.
